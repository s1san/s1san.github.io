几何漫谈：随手记与草稿堆

Hi👋，这里是几何漫谈😀

这是一个持续更新的科普项目，内容目前以数学（几何）为主，用可视化表达直观理解世界。

随手记

我使用 Typora / VS Code 编辑文案，Python + Manim + VS Code 构建动画，剪映 制作视频。
